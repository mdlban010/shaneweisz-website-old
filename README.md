# Shane Weisz Website

This site has been rebuilt using [Next.js](https://nextjs.org/).

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start a development server:
   ```bash
   npm run dev
   ```

## Production Build

Run the following to create an optimized production build:

```bash
npm run build
```

Then start the app with:

```bash
npm start
```
